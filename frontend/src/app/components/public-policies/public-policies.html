POLITICAS

Politicas de uso y operacion

Estas politicas rigen el uso de la plataforma para mantener seguridad, continuidad y control operativo.

1. Control de acceso y autenticacion

Todos los usuarios deben operar con credenciales personales. No se permite compartir cuentas entre miembros del equipo.

Los permisos estan definidos por rol y deben respetarse durante toda la operacion.

2. Proteccion de datos

La informacion de clientes, creditos y pagos se considera confidencial y solo puede usarse para fines operativos autorizados.

Queda prohibida la extraccion o divulgacion no autorizada de datos.

3. Registro de acciones

Las acciones de cobranza deben quedar registradas para mantener trazabilidad y soporte en auditorias internas.

La edicion de registros debe seguir control administrativo y justificacion operativa.

4. Continuidad del servicio

Las solicitudes de cambio, incidencias o nuevas funciones se gestionan por el canal oficial del sistema.

El incumplimiento de estas politicas puede implicar suspension de acceso segun lineamientos internos.
